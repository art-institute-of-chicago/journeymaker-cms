Field Detail
allDay
Set to TRUE if this is an all day event
category

The category of this event.

  • Game
  • Meeting
  • Other
  • Party
  • Practice
  • Scrimmage

Game and Scrimmage categories require competitors array to be passed and will override the title.

competitors
An associative array of competitor information, where the key is the UUID of the competitor and each entry contains a label and score like the following. 

  var competitors = {
    '123456789' => {
      'label':'Competitor 1',
      'score':5
    },
    '232342342' => {
      'label':'Competitor 2',
      'score':10
    }
  };
